# CancerDriverEngine

Cancer driver gene identification engine using dN/dS analysis, hotspot detection, and oncogene/tumor suppressor classification.

## Features

- dN/dS ratio calculation for positive selection detection
- Mutation hotspot identification and statistical testing
- Oncogene and tumor suppressor gene classification
- Pan-cancer driver gene landscape analysis
- Functional impact scoring and pathway enrichment

## Results

500 genes × 200 tumors; 73 genes dN/dS>2; 54 hotspots; 13 oncogenes, 17 TSGs

## Usage

```bash
pip install numpy scipy matplotlib
python cancer_driver_engine.py
```
